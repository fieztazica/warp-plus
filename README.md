# warp-plus

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run src/index.ts
```

To use as cli:
```bash
bun link
bun link warp-plus

# Use
wp --help
```

This project was created using `bun init` in bun v1.1.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
